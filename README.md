# -
Описание
Telegram-бот для техникума, разработанный как медицинский помощник. Проект активно развивается и будет регулярно обновляться.

Текущие Возможности
Главное Меню: Удобная навигация по функциям.
Поиск Лекарств: Информация о медикаментах и аналогах.
Экстренная Помощь: Контакты и советы в чрезвычайных ситуациях.
Напоминания: Установка напоминаний о приёме лекарств.
Дневник Самочувствия: Записи о настроении, симптомах и заметках.
Технологии
Язык: Python 3.14.2
Библиотека: aiogram
Управление диалогами: FSM (Finite State Machine)
Хранение данных: Временное (в памяти)
Создайте и активируйте виртуальное окружение:
в cmd
python -m venv venv
    # Windows: .\venv\Scripts\activate
    # macOS/Linux: source venv/bin/activate
Установите зависимости
в cmd
pip install -r requirements.txt
    # Создайте requirements.txt с содержимым: aiogram==3.*
Получите токен бота от @BotFather в Telegram.
Запустите бота
